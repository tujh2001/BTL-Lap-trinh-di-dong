## Ứng dụng quản lý lịch cá nhân

Tính năng: Ứng dụng này sẽ giúp bạn kiểm soát các lịch trình của mình, từ đó nâng cao năng suất làm việc của mình.

- Phát triển dựa trên:  [expo](https://expo.io).

### Cài đặt

```
yarn install
```
